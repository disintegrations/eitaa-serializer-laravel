eitaa serializer laravel package

Use Eitaa TL methods directly from Laravel.

Serialize Telegram/Eitaa-style TL method calls, wrap them in an Eitaa gateway object, send the binary payload with Laravel HTTP, and deserialize the binary response.

Get started View examples

->

->

->

Installation

Use the package through Composer. Laravel package discovery registers the service provider automatically.

bash 
composer require disintegrations/eitaa-serializer-laravel

Requirements

Composer will enforce these constraints.

php ^8.3 illuminate/http ^12|^13 illuminate/support ^12|^13 ext-json ext-zlib

Configuration

The default configuration works for normal use. Publish files only when you need to override the endpoint, layer, timeout, IMEI, or schema path.

bash 
php artisan vendor:publish --tag=eitaa-config
.env 
EITAA_GATEWAY_ENDPOINT=https://sajad.eitaa.ir/eitaa/
EITAA_LAYER=133
EITAA_DEFAULT_IMEI=00__web
EITAA_TIMEOUT=30

Publish the bundled schema only when it must be customized.

bash 
php artisan vendor:publish --tag=eitaa-schema
.env 
EITAA_SCHEMA_PATH=/absolute/path/to/resources/eitaa/schema.json

Usage

Resolve the client from the container, inject it into your service, or use the facade.

Container client

php 
use Disintegrations\EitaaSerializer\EitaaGatewayClient;

$response = app(EitaaGatewayClient::class)->send(
    method: 'help.getConfig',
    params: [],
    token: null,
    imei: null,
);

Facade

php 
use Disintegrations\EitaaSerializer\Facades\Eitaa;

$response = Eitaa::send('help.getConfig');

Authenticated method

php 
$response = app(EitaaGatewayClient::class)->send(
    method: 'messages.sendMessage',
    params: [
        'flags' => 0,
        'peer' => [
            '_' => 'inputPeerUser',
            'user_id' => '123456789',
            'access_hash' => '987654321',
        ],
        'message' => 'Hello',
        'random_id' => (string) random_int(1, PHP_INT_MAX),
    ],
    token: $token,
    imei: $imei,
);

TL Value Rules

The serializer follows the bundled TL schema. Shape request arrays to match method params and constructor predicates.

Value Rule Example
Object constructor Use `_` for the predicate. ['_'=> 'inputPeerSelf']
Optional fields Set the required `flags` bit or the field is omitted. 'flags' => 1
long Use strings when values can exceed PHP integer range. 'random_id' => '123456789'
bytes and intN Use raw binary strings or arrays of byte integers. hex2bin('6b18f9c4')
If a request fails while serializing, check the schema entry for the method, nested constructor predicates, and exact flag bits.

Testing

The package includes fast unit tests and live integration tests for no-auth Eitaa methods.

Normal tests

No real Eitaa network calls.

bash 
composer install
composer test

Live integration tests

Calls no-auth methods on the Eitaa gateway.

bash 
EITAA_RUN_INTEGRATION=1 composer test:integration
powershell 
$env:EITAA_RUN_INTEGRATION='1'; composer test:integration

Troubleshooting

Class not found

Run Composer autoload refresh and confirm package discovery has registered the provider.

bash 
composer dump-autoload
php artisan package:discover

Config values stale

Clear Laravel's cached configuration after changing `.env` or published config.

bash 
php artisan config:clear

Deserializer error

Dump the response status, content type, length, and first bytes in hex before parsing.

php 
dd([
    'status' => $response->status(),
    'content_type' => $response->header('content-type'),
    'length' => strlen($raw),
    'first_64_hex' => bin2hex(substr($raw, 0, 64)),
]);
